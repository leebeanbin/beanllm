## ✅ 권장: 옵션 D (하이브리드)

**레이아웃**:
```
┌─────────────────────────────────────────────────────┐
│ 입력창                                               │
│ ┌───────────────────────────────────────────────┐  │
│ │ 메시지를 입력하세요...                          │  │
│ └───────────────────────────────────────────────┘  │
│                                                     │
│ [📎 파일] [🖼️ 이미지] [🎤 음성] [📊 데이터]        │
│                                                     │
│ 자동 모드 (기본) | [⚙️ 수동 선택]                  │
└─────────────────────────────────────────────────────┘
```

**특징**:
- 기본: 자동 모드 (모든 기능 자동 감지)
- 수동 선택 모드: 토글 버튼으로 전환
- 파일 업로드: 버튼으로 명확히 표시
